En el archivo LexAsgard.x se definen las expresiones regulares que pueden estar contenidas en el lenguaje. Asímismo, se presenta una la definición de los tokens, con la acción para cada expresión regular. 

El archivo Tokens.hs contiene la lista de tokens de AsGArD con sus argumentos, si aplica, y el String asociado que se imprimirá al hacer el análisis lexicográfico. 

El archivo LiteralCanvas.hs contiene funciones que permiten, en función del literal lienzo recibido, imprimir el token correspondiente. 

Se distribuyó el código de esa manera para mantener la modularidad del programa y facilitar cualquier modificación sobre el mismo. 

Para ejecutar el programa, es necesario ejecutar:

$ make
$ cat archivo | ./LexAsgard

En caso de fallar el make, hacer:

$ make alex
$ make ghc
$ cat archivo | ./LexAsgard
